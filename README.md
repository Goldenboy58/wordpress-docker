# WordPress Docker Setup

This repository contains a Docker Compose configuration to run **WordPress** with a **MySQL database** locally. It’s designed for quick setup, testing, and development.

## 🚀 Getting Started

### Prerequisites
- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/)

### Installation
